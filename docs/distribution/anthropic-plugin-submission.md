## Status policy

This document may be changed to `Submitted` only after Anthropic returns a
submission acknowledgement. The repository may state `Listed in
claude-community` only after Codex Orchestrator appears in the public
`anthropics/claude-plugins-community` catalog.

`claude-plugins-official` is curated separately at Anthropic's discretion.
There is no application process, and a community submission must never be
described as an application to or approval by the official catalog.

The first-party GitHub marketplace remains independently installable while an
Anthropic review is pending or if Anthropic declines the submission.
